# 截图OCR提取工具

一个零安装的网页工具，用于上传截图并自动提取其中的中文和英文文本。**纯前端运行，无需安装任何软件！**

## 🎯 两个版本供选择

### 版本1: 纯前端版 (推荐 ✅)
**零安装，打开网页直接使用！**
- 🌐 只需双击 `index.html` 在浏览器中打开
- 🚫 无需安装 Python、Tesseract 或其他依赖
- ⚡ 完全在浏览器中运行，使用 Tesseract.js
- 📱 支持离线使用（需提前加载语言包）

### 版本2: 后端版 (备选)
使用 Flask + Tesseract OCR，适合需要本地处理大量图片的场景。

## ✨ 功能特点

- 🚀 **零安装** - 纯前端运行，打开即用
- 📋 **一键粘贴** - 截图后直接Ctrl+V粘贴，自动提取文字
- 🖼️ **多格式支持** - PNG、JPG、JPEG、BMP
- 🔤 **中英文识别** - 自动识别混合文字
- 📋 **一键复制** - 结果直接复制到剪贴板
- ⚡ **自动提取** - 粘贴后自动开始OCR处理
- 📱 **响应式设计** - 适配电脑和手机浏览器
- 🎯 **进度显示** - 清晰的OCR处理进度

## 🌐 纯前端版使用指南

### 使用方法：快捷粘贴（唯一操作方式）
1. **打开工具**：双击 `index.html` 文件
2. **截图**：使用系统截图工具（Windows: `Win+Shift+S`, macOS: `Cmd+Shift+4`）
3. **粘贴**：回到工具页面，按 `Ctrl+V` 粘贴截图
4. **自动提取**：图片自动上传并开始提取文字（需开启"自动提取"选项）
5. **获取结果**：等待处理完成，点击"复制"按钮复制结果

### 备用粘贴方式
- **手动粘贴按钮**：点击"粘贴截图并提取文字"按钮手动粘贴
- **语言选择**：在粘贴前可勾选需要识别的语言（默认中英文都选中）
- **自动提取选项**：可开启/关闭"粘贴后自动开始提取"选项

### 技术原理
- 使用 [Tesseract.js](https://github.com/naptha/tesseract.js) 在浏览器中运行OCR
- 通过CDN加载OCR引擎和语言包
- 所有处理都在本地浏览器完成，不上传图片到服务器

### 注意事项
- 首次使用需要加载语言包（约10-20MB），请耐心等待
- 需要网络连接加载OCR引擎（仅第一次需要）
- 图片越清晰，识别准确率越高
- 支持文件大小不超过10MB
- **粘贴功能**：在页面任意位置按 `Ctrl+V` 即可粘贴截图
- **自动提取**：默认开启，粘贴后自动开始OCR处理

## 🖥️ 后端版使用指南

如果您需要处理大量图片或需要更高性能，可以使用后端版本：

### 安装步骤
1. **安装 Tesseract OCR**
   - Windows: [下载安装包](https://github.com/UB-Mannheim/tesseract/wiki)（记得勾选中文语言包）
   - macOS: `brew install tesseract tesseract-lang`
   - Linux: `sudo apt-get install tesseract-ocr tesseract-ocr-chi-sim`

2. **安装 Python 依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **启动服务**
   ```bash
   python app.py
   ```

4. **打开浏览器访问** `http://127.0.0.1:5000`

## ❓ 常见问题

### Q: 纯前端版需要网络连接吗？
**A:** 首次使用时需要网络连接加载Tesseract.js引擎和语言包（约10-20MB）。加载完成后，后续使用可以离线进行。

### Q: 为什么第一次使用比较慢？
**A:** 首次使用需要从CDN下载OCR引擎和语言包，请耐心等待。下载后会缓存在浏览器中，后续使用会很快。

### Q: 提取的文字不准确怎么办？
**A:** OCR识别准确率受图像质量影响：
   - 使用清晰、高分辨率的截图
   - 确保文字在图像中足够大
   - 避免背景复杂或对比度低的图片
   - 尝试调整语言选择（只选中文或英文）

### Q: 支持哪些浏览器？
**A:** 支持现代浏览器（Chrome 80+、Firefox 75+、Edge 80+、Safari 14+）。**注意：** Safari可能需要启用相关权限。

### Q: 图片会传到服务器吗？
**A:** **不会！** 纯前端版所有处理都在您的浏览器本地完成，图片不会上传到任何服务器。

### Q: 如何处理大量图片？
**A:** 如果需要批量处理大量图片，建议使用后端版本（`app.py`），性能更好且不受浏览器内存限制。

### Q: 如何离线使用？
**A:** 确保在联网状态下打开一次工具，加载完语言包后，就可以离线使用了。

### Q: 粘贴功能怎么用？
**A:** 截图后，在工具页面按 `Ctrl+V` 即可粘贴。粘贴后工具会自动提取文字（如果开启了"自动提取"选项）。您也可以点击"粘贴截图并提取文字"按钮手动粘贴。

## 🛠️ 技术栈

### 纯前端版
- **OCR引擎:** [Tesseract.js](https://github.com/naptha/tesseract.js) (WebAssembly)
- **前端框架:** 原生HTML/CSS/JavaScript
- **剪贴板处理:** Clipboard API + Paste 事件
- **图像处理:** FileReader API + Blob 处理
- **UI组件:** 自定义响应式设计，专注粘贴体验

### 后端版
- **后端框架:** Flask (Python)
- **OCR引擎:** Tesseract (原生)
- **图像处理:** Pillow (PIL)
- **前端界面:** 原生HTML/CSS/JavaScript
